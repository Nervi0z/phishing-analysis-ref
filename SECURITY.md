# Security Policy

## Scope

This repository contains documentation, tool references, and analysis workflows. It does not include executable code or deployed infrastructure.

## Usage notice

All tools and techniques documented here are for authorized defensive use — your own infrastructure, authorized penetration tests, or purpose-built lab environments. The simulation section documents offensive tooling for educational purposes only. Using any of these tools against systems without explicit written authorization is illegal.
